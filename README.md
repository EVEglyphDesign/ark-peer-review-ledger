# ARK Peer Review Ledger

A source-first, cataloguing surface for **Shawn Ryan Show #320 with Danny Goler** —
the episode where the conversation, around the one-hour mark, lands on the alignment
test, competition-to-collaboration, and the code beneath the reveal.

This surface exists so that a listener who came off the Spotify posting can find
where the shape they heard has already been written down, technically, in the
working ledger canon of EVE Glyph Design.

## What it is

- A static site (GitHub Pages) under the EVE Glyph Design canon.
- An **entry gate** — three doors: Listener, Peer, Institution — pure client-side.
- A **Peer Review Ledger** — timestamps × canon cross-references × resonance score.
- An **anonymous signal form** — composes a public GitHub issue in the visitor's
  browser. Nothing is transmitted from the page. No accounts. No cookies. No analytics.

## Posture

- Standard-intake only. No scraping, no back channels, no adversarial outreach.
- Safety first, betterment second.
- Non-disruptive: incumbents (podcast, guest, platforms) keep their operational role.
- No profiling. No listener is scored, categorised, or targeted.
- The theorists have pieces of the puzzle. The ledger holds the shape.

## Files

- `index.html` — the surface itself.
- `assets/style.css`, `assets/portal.css` — inherited EVE Glyph Design canon
  (from `paix-parish-platform`).
- `assets/ledger.css` — extensions specific to this surface.
- `assets/portal-crest.svg` — shared crest.
- `.nojekyll` — GitHub Pages static passthrough.
